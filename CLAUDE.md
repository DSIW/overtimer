## Agent skills

### Issue tracker

Issues are tracked in GitHub Issues for DSIW/overtimer via the `gh` CLI; external PRs are also pulled into the triage queue as a request surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Default label vocabulary (needs-triage, needs-info, ready-for-agent, ready-for-human, wontfix) — `wontfix` already matches your existing repo label. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context layout: `CONTEXT.md` + `docs/adr/` at the repo root. See `docs/agents/domain.md`.
